Detailed Description

Performs an encoding analysis operating on multidimensional cochleograms extracted for each of the two audio channels. For computational purposes we here reduce the dimensionality of this audio representation using principal component analysis.

Parameters
subidsubject id (an integer between 1 and 44)
bidsdirthe root of the BIDs directory
sdirdirectory in which the features and model results are stored
eegpreproca flag (0/1) that indicates whether or not to build eeg features. 1 = assumes that these features have already been extracted
audpreproca flag (0/1) that indicates whether or not to build audio features. this can by handy with limited signal processing licenses
fitmodelsa flag (0/1) that indicates whether or not to actually fit and evaluate the encoding models
