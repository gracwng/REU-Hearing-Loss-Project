function regevalnestedloopssvdridge (in opts, in xx, in yy)
 

Detailed Description

Fits Ridge regression models using <regfitsvdridge> and evaluates the predictive power using nested cross-validation procedures.

Parameters
optsconfiguration struct
opts.lambdaRidge regularization hyperparameters of interest. Default: [10 100 1000]
opts.znormdShould the data be Z-scored?. Default: 1 (yes). Alternatively set it to 0 (no)
opts.gofWhat goodness-of-fit metric is considered? default: 'estcorr'. Please note that this is an advanced field and should most often be left as is.
opts.kouterNumber of outer folds. Default: 10
opts.kinnerNumber of inter folds. Default: 5
opts.storewDo you wish to store the model coefficients? Default: 0 (no), alternatively set it to 1 (yes)
opts.phaseranda configuration struct for evaluating model performance on surrogate data
opts.phaserand.npermnumber of times to evaluate goodness-of-fit on surrogate data for each outer fold
opts.phaserand.roishould the goodness-of-fit be averaged over certain columns in yy? Default was for this project set to a cluster of fronto-central electrodes
xxregressor matrix of size (time x features x trials)
yytarget matrix of size (time x features x trials)
Return values
vloga cellstruct containing information about model fit and performance
vlog{i}.accimodel accuracy over each inner loop < number of lambda paramaters x dimensions in yy x inner folds>
vlog{i}.accomodel accuracy on the test data for outer loop i
vlog{i}.optsa structure containing information about <opts> and the cross-validation splits
vlog{i}.opts.phaseranda structure containing model accuracies on surrogate data
vlog{i}.opts.phaserand.accsurrmodel accuracies over all permutations
vlog{i}.opts.phaserand.roiinput roi specified. this may however not be fulfilled, in which case it is set to the average over all dimensions in yy
vlog{i}.opts.phaserand.surroithe actual roi considered for the analysis
