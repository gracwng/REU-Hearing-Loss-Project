Detailed Description

Function used for exporting data summaries and figures.

Parameters
sdirdirectory in which the features and model results are stored
bidsdirthe root of the BIDs directory
do_plota flag (0/1) indicating if relevant figures should be exported (1) or not (0)
cameqspecify if results should be exported with- or without CamEQ. this can be 'wa' (with CamEQ), 'woa' (without CamEQ; extracted using inverse filtering) or 'woacontrol' (without CamEQ; exactly the same audio extraction method for NH and HI listeners)
savesummaryshould the data summary be stored in ds-nhhi>reports? 1 = yes, 0 = no
