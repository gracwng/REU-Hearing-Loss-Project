\subsection{Detailed Description}
Resamples a 2d array, but first centers the array and afterwards adds the mean to the output. The function outputs the resampled array and a corresponding time vector is generated (relative to a pre-\/specified start time $<$ti$>$) 


\begin{DoxyParams}{Parameters}
{\em xx} & array (time x independent dimensions) \\
\hline
{\em fsr} & sampling rate after resampling \\
\hline
{\em fs} & original sampling rate \\
\hline
{\em ti} & a number indicating the time of the first sample. the corresponding generated time vector will be relative to this sample \\
\hline
\end{DoxyParams}
