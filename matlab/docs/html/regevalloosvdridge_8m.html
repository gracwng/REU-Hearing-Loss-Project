function regevalloosvdridge (in opts, in xx, in yy)
 

Detailed Description

Fits Ridge regression models using <regfitsvdridge> and evaluates the predictive power using a leave-one-trial-out cross-validation technique.

Parameters
optsconfiguration struct
opts.lambdaRidge regularization hyperparameters of interest. Default: [10 100 1000]
opts.znormdShould the data be Z-scored?. Default: 1 (yes). Alternatively set it to 0 (no)
opts.gofWhat goodness-of-fit metric is considered? default: 'estcorr'. Please note that this is an advanced field and should most often be left as is.
xxregressor matrix of size (time x features x trials)
yytarget matrix of size (time x features x trials)
Return values
wwmodel weights for the model that yield best goodness-of-fit during cross-validation
accprediction accuracy after averaging over all outer folds for each lambda and selecting the maximum (please note that this may show signs of overfitting and should thus be taken with a grain of salt, and we did not use this measure)
