function preproc_eogdenoise (in opts, in dat)
 

Detailed Description

Regress out EOG artifacts from EEG data (stored in Fieldtrip format)

Parameters
optsconfiguration structure
opts.demeana 0/1 flag, 1 = center the data prior to spatial filtering (default), 0 = leave as is
opts.zthreshwhat threshold do we set for the eog artefact detection? default = 4
opts.eogchanwhich eog channels do we here consider? e.g. {'EOG'}
opts.pthreshset a power ratio threshold that defines how many eigenvectors to retain
opts.artsegthis is handy if you want to manually label EOG artifacts (which is fairly straightforward). when this field is not empty, Fieldtrip will not be used for EOG labelling
data Fieldtrip data structure
Return values
outa structure with relevant fields for denoising (filters and z-scoring parameters)
datdenoised eeg data (in Fieldtrip format)
