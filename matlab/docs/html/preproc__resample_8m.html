Detailed Description

Resamples a 2d array, but first centers the array and afterwards adds the mean to the output. The function outputs the resampled array and a corresponding time vector is generated (relative to a pre-specified start time <ti>)

Parameters
xxarray (time x independent dimensions)
fsrsampling rate after resampling
fsoriginal sampling rate
tia number indicating the time of the first sample. the corresponding generated time vector will be relative to this sample
