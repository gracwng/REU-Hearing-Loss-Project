function sr_classificationanalysis (in opts, in srdat)
 

Detailed Description

Trains a reconstruction model on single-talker data using LOO and evaluates how accurately the model can be used to discrinate between attended and unattended speech from two-talker data.

Parameters
optsconfiguration structure
opts.tdurduration of decoding segments (in s)
opts.fssampling rate of preprocessed data (e.g. fs=64)
opts.tshifthow long are the shifts (in s) between each segment? the decoding segments are nonoverlapping and shifted with opts_classification.tdur + opts_classification.tshift)
opts.lagoilags of interest (e.g. -32:0)
opts.znormdflag indicating if the data should be z-scored (1) or not (0)
opts.lambdaregularization parameters of interest (e.g [1 10 100])
opts.gofgoodness of fit metric (e.g. 'estcorr')
srdatimported and preprocessed audio/EEG data (see sr_importdata.m)

The backward models in this project are based on ridge-regularized filters (and not tikhonov filters as they potentially introduce relatively strong cross-channel leakages)
