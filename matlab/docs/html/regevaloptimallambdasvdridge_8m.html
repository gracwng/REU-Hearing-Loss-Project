Detailed Description

Fits Ridge regression models using <regfitsvdridge> with fixed regularization parameters and optionally assesses predictive power on indepent data.

Parameters
xxregressor matrix of size (time x features)
yytarget matrix of size (time x features)
xtvalidation (or test) regressor matrix (time x features)
ytvalidation (or test) target matrix (time x features)
optindexindexes indicating which regularization parameters to consider for each output dimension. If the output from a cross-validation analysis e.g. suggests that lambda=0.1 is optimal for voxel/channel 1 and lambda=10 is optimal for voxel/channel 2 then optindex = [1 3] if lambda=[0.1 1 10]
lambdaregulariation parameters considered
znormda flag indication whether or not the data is zscored
gofgoodness-of-fit metric to be considered (default: 'estcorr')
Return values
accgoodness of fit (e.g. correlation coefficient)
wregression coefficients (number of dimensions in xt x number of dimensions in yt)
